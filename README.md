# 🛍️ LUXE – Full Stack E-Commerce Platform

LUXE is a modern full-stack e-commerce application built using React for the frontend and Django REST Framework for the backend. The project provides secure user authentication, product browsing, shopping cart management, and order processing.

---

# ✨ Features

## 👤 User

- User Registration
- Secure Login
- JWT Authentication
- Product Browsing
- Search Products
- Add to Cart
- Remove from Cart
- Checkout

---

## 🛒 Shopping Cart

- Add Products
- Remove Products
- Update Quantity
- View Total Price

---
